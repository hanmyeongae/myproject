프롬프트 예시
사용자 프롬프트:

제목: JPG 이미지를 PDF로 변환하는 웹 앱을 만들어줘

내용:

목표: 사용자가 JPG 파일을 업로드하면, 이를 PDF 파일로 변환하고 변환된 파일을 다운로드할 수 있는 간단한 웹 애플리케이션을 만들어줘.

기술 스택:

HTML: 웹 페이지의 기본 구조를 만듭니다.

CSS: 웹 페이지를 보기 좋게 스타일링합니다.

JavaScript: 파일 업로드, 변환, 다운로드 기능을 담당하는 핵심 로직을 구현합니다. 외부 라이브러리인 **jsPDF**를 사용하여 이미지를 PDF로 변환하는 기능을 추가해줘.

기능 요구사항:

파일 업로드: 사용자가 JPG 파일을 선택할 수 있는 <input type="file"> 요소를 포함해야 합니다.

변환 버튼: 사용자가 파일을 선택한 후 변환을 시작할 수 있는 버튼이 있어야 합니다.

진행 상황: 변환이 진행 중임을 사용자에게 알리는 간단한 메시지나 로딩 상태를 표시해야 합니다.

다운로드: 변환이 완료되면, 변환된 PDF 파일을 자동으로 다운로드하거나 다운로드 링크를 제공해야 합니다.

오류 처리: JPG 파일이 아닌 다른 형식의 파일이 업로드되었을 경우, 사용자에게 경고 메시지를 표시해야 합니다.

코드 구조:

하나의 HTML 파일(index.html)에 모든 코드(HTML, CSS, JavaScript)를 포함하는 형태로 만들어줘. 외부 파일로 분리하지 않아도 돼.

코드에 각 기능이 어떤 역할을 하는지 자세한 주석을 달아줘.

라이브러리 사용을 위해 CDN 링크를 포함시켜줘.
